# Getting Started

## Required Libraries and Frameworks

- Node 16.x
- JDK 17.0.x
- (Android) Android Studio SKD
- yarn

# Running

- Open a virtual device
- Install

```
yarn install
```

- Run Metro

```
yarn start
```
